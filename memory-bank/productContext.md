# Product Context - Lumigen

## Why This Project Exists
Lumigen was created to provide a local, private alternative to cloud-based AI image generation services. Users can run their own image generation with various AI providers while maintaining full control over their data.

## Problems It Solves
1. **Privacy Concerns**: Users don't want to upload prompts/images to third-party services
2. **Provider Flexibility**: Support multiple AI providers (OpenAI, OpenRouter, Google, BFL)
3. **Session Management**: Track and manage multiple generation sessions
4. **Asset Organization**: Categorize and filter generated images
5. **Profile Management**: Save and reuse generation configurations

## How It Should Work
- Fast, responsive web interface with HTMX for SPA-like experience
- Clear session history with time-based categorization
- Intuitive gallery with click-to-select for bulk operations
- Robust admin panel for provider configuration

## User Experience Goals
- Dark theme optimized for long sessions
- Quick access to recent generations
- Easy bulk operations in gallery (click to select)
- Clear feedback on generation status
- Responsive design for various screen sizes
